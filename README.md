# toga-leet-code
### Instructions:
### What needs to be done to activate the system:
1. open cmd
2. run  ``` git clone git@github.com:tzivia-rot/toga-leet-code.git ```
3. #### to run server:
     1. run ```cd server```
     2. run ```docker-compose up```
4. #### to run client - cmd:
   1. run ```cd command-line```
   2. rum ```go run main.go```
